## Цель работы
Разработать расширение для VS Code для автоматической вставки текущей даты и времени в текст.

## Выполненная работа

### 1. Разработано расширение "Timestamp Inserter"
**Функционал:**
  - Отображение текущего времени в статус-баре VS Code
  - Автообновление времени каждые 60 секунд
  - 3 формата вставки: время, дата, полная дата-время
  - Интеграция в контекстное меню редактора

### 2. Техническая реализация
- **Язык:** TypeScript/JavaScript
- **Объём кода:** ~100 строк
- **Архитектура:** Модульная структура с разделением логики
- **Сборка:** TypeScript Compiler (tsc)

### 3. Результат
✅ Расширение успешно устанавливается и работает в VS Code
✅ Время отображается в статус-баре и обновляется автоматически
✅ Реализовано меню выбора форматов через QuickPick
