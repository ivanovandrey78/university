# Changelog

## [0.1.0] - 18-11-2025

### Added
- Первоначальная структура плагина, созданная с помощью генератора Yeoman
- Базовая настройка расширения TypeScript

---

## [0.1.1] - 19-11-2025

### Added
- Ручная настройка структуры плагина после проблем с Yeoman
- Базовые конфигурационные файлы: `package.json`, `tsconfig.json`, `launch.json`
- Первая рабочая команда "Hello World" для проверки функциональности

### Fixed
- Проблемы с установкой и запуском TypeScript компилятора
- Ошибки валидации `package.json` (отсутствующие обязательные поля `version` и `engines`)
- Настройка правильной конфигурации отладки для VS Code расширений

### Changed
- Переход с автоматической генерации на ручную настройку проекта
- Упрощение структуры проекта до минимально необходимых файлов

### Technical Details
- Настроена компиляция TypeScript в JavaScript через `npm run compile` - написан скрипт для быстрой компиляции / запуска программы
- Реализована система активации плагина по команде через `activationEvents`
- Добавлена базовая интеграция с VS Code API (команды и уведомления)

---
