# Changelog

## [0.1.0] - 2025-11-21
### Добавлено
- Статус-бар с текущим временем в нижней панели VS Code
- Автообновление времени каждые 60 секунд
- Интеграция в контекстное меню редактора (правый клик → Insert Timestamp)

### Изменено
- Улучшена обработка ошибок при отсутствии активного редактора

### Исправлено
- Исправлен баг форматирования минут (теперь `09` вместо `9`)

## [0.0.2] - 2025-11-20
### Добавлено
- 3 формата вставки времени:
  - `HH:MM` — только время (14:30)
  - `YYYY-MM-DD` — дата в ISO формате (2025-12-12)
  - `DD.MM.YYYY HH:MM:SS` — полная дата и время (10-02-1163 14:30:45)
- Базовая интеграция в Status Bar VS Code
- Команда `timestamp.insert` для вызова из Command Palette (Ctrl+Shift+P)


## [0.0.1] - 2025-11-19
### Добавлено
- Инициализация проекта TypeScript
- Базовая конфигурация `package.json` с метаданными расширения
- Прототип функции вставки текущего времени
- Система сборки через TypeScript Compiler (tsc)
