#### **Кузнечик** (jump frog)

- Кузнечик прыгает по столбикам. Каждый столбик имеет стоимость (энергия). Нужно дойти до конца с минимальной суммой.
    
- Можно прыгать на 1 или 2 столбика.
    

**Подзадача:**
```cpp
dp[i] = минимальная энергия, чтобы добраться до i-го столбика
```

**Рекуррентная формула:**
```cpp
dp[i] = min(dp[i-1] + cost[i-1 → i], dp[i-2] + cost[i-2 → i])
```

# **1️⃣ Кузнечик**

**Условие:**

- Кузнечик прыгает по столбикам. Каждый столбик имеет стоимость.
    
- Прыгать можно на 1 или 2 столбика.
    
- Нужно дойти до конца с минимальной суммой энергии.
    

**Данные:**

```
столбики:   0   1   2   3   4
стоимость:  0   2   3   1   4
```

**Инициализация dp:**

```
dp[0] = 0  (старт)
```

**Шаги:**

1. i = 1 → min(dp[0] + cost(0→1)) = 0 + 2 = 2
    
    ```
    dp = [0, 2, -, -, -]
    ```
    
2. i = 2 → min(dp[1]+cost(1→2)=2+3=5, dp[0]+cost(0→2)=0+3=3)
    
    ```
    dp = [0, 2, 3, -, -]
    ```
    
3. i = 3 → min(dp[2]+cost(2→3)=3+1=4, dp[1]+cost(1→3)=2+1=3)
    
    ```
    dp = [0, 2, 3, 3, -]
    ```
    
4. i = 4 → min(dp[3]+cost(3→4)=3+4=7, dp[2]+cost(2→4)=3+4=7)
    
    ```
    dp = [0, 2, 3, 3, 7]
    ```
    

**Ответ:** минимальная энергия = 7

**Визуализация “шагов”:**

```
столбики:    0   1   2   3   4
стоимость:   0   2   3   1   4
dp:          0   2   3   3   7
```

- Каждое dp[i] = минимальная энергия, чтобы добраться до i-го столбика
