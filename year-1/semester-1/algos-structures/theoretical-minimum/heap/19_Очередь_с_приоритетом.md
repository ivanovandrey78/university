## 19. Очередь с приоритетом (повышение / понижение приоритета)  

### Суть структуры

**Очередь с приоритетом (Priority Queue)** — это абстрактная структура данных, в которой каждый элемент имеет **приоритет**,  
и операции выполняются не в порядке добавления (как в обычной очереди), а по **наивысшему приоритету**.

Чаще всего реализуется через **бинарную кучу (heap)**:

- В **max-heap** — элемент с наибольшим приоритетом находится в корне.
    
- В **min-heap** — элемент с наименьшим приоритетом в корне.

### Основные операции

|Операция|Описание|Сложность|
|---|---|---|
|**insert(x)**|Вставка нового элемента|`O(log n)`|
|**extract_max() / extract_min()**|Удаление элемента с наибольшим/наименьшим приоритетом|`O(log n)`|
|**peek()**|Просмотр корня (элемента с максимальным приоритетом)|`O(1)`|
|**increase_key(i, new_value)**|Повышение приоритета элемента|`O(log n)`|
|**decrease_key(i, new_value)**|Понижение приоритета элемента|`O(log n)`|
### Идея изменения приоритета

Изменение приоритета элемента `arr[i]` может **нарушить свойство пирамиды**,  
поэтому требуется восстановление структуры:

1. **Если приоритет повысился** → элемент стал "больше", чем родитель → вызываем `sift_up(i)`.
    
2. **Если приоритет понизился** → элемент стал "меньше", чем потомки → вызываем `heapify_down(i)`.



### Реализация

```cpp
void sift_up(int* heap, int i) {
    while (i > 0 && heap[(i - 1) / 2] < heap[i]) {
        std::swap(heap[i], heap[(i - 1) / 2]);
        i = (i - 1) / 2;
    }
}

void heapify_down(int* heap, int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;

    if (left < n && heap[left] > heap[largest])
        largest = left;
    if (right < n && heap[right] > heap[largest])
        largest = right;

    if (largest != i) {
        std::swap(heap[i], heap[largest]);
        heapify_down(heap, n, largest);
    }
}

// Повышение приоритета (увеличение ключа)
void increase_key(int* heap, int i, int new_value) {
    if (new_value < heap[i]) return; // нельзя уменьшать приоритет в этой функции
    heap[i] = new_value;
    sift_up(heap, i);
}

// Понижение приоритета (уменьшение ключа)
void decrease_key(int* heap, int n, int i, int new_value) {
    if (new_value > heap[i]) return; // нельзя повышать приоритет в этой функции
    heap[i] = new_value;
    heapify_down(heap, n, i);
}
```



### Асимптотика

|Операция|Время|
|---|---|
|Вставка|`O(log n)`|
|Извлечение максимума|`O(log n)`|
|Просмотр максимума|`O(1)`|
|Повышение/понижение приоритета|`O(log n)`|
|Построение из массива|`O(n)`|

### Применение

- Планировщики задач (CPU Scheduler)
    
- Алгоритмы Дейкстры и Прима
    
- Системы событий (event-driven simulation)
    
- Реализация `std::priority_queue` в C++ STL
