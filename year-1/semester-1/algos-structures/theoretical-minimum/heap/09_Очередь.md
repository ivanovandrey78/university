## 9. Очередь  

### Определение

**Очередь (Queue)** — это абстрактная структура данных, работающая по принципу  
**FIFO (First In — First Out)** — _первый вошёл, первый вышел._

> Пример: очередь в магазине — кто первый пришёл, тот первый обслужен.

### Основные операции

| Операция           | Описание                              | Время    | Память   |
| ------------------ | ------------------------------------- | -------- | -------- |
| `push(x)`          | добавить элемент `x` на вершину стека | **O(1)** | **O(1)** |
| `pop()`            | удалить верхний элемент               | **O(1)** | **O(1)** |
| `front()`          | вернуть верхний элемент (не удаляя)   | **O(1)** | **O(1)** |
| `empty()`          | проверить, пуст ли стек               | **O(1)** | **O(1)** |
| `size()`           | вернуть количество элементов          | **O(1)** | **O(1)** |
| `find(x)`          | найти элемент x в очереди             | **O(n)** | **O(1)** |


### Использование очереди

| Применение                  | Пример                              |
| --------------------------- | ----------------------------------- |
| Обработка задач по порядку  | планировщик процессов               |
| BFS (поиск в ширину)        | алгоритмы на графах                 |
| Буферизация данных          | ввод-вывод, потоки                  |
| Очереди событий             | обработка входных сообщений         |
| Симуляция реальных очередей | моделирование касс, клиентов и т.д. |
|                             |                                     |

## **2️⃣ Очередь (Queue) — FIFO**

```
enqueue(10) → enqueue(20) → enqueue(30)

Фронт → 10
         20
Зад    → 30
```

**Операции:**

| Операция | Что делает             | Сложность                |
| -------- | ---------------------- | ------------------------ |
| enqueue  | добавляет в конец      | O(1)                     |
| dequeue  | удаляет из начала      | O(1) с кольцевым буфером |
| front    | смотрит первый элемент | O(1)                     |

---
## **3️⃣ Дек (Deque) — двусторонняя очередь**

```cpp
push_front(1) → push_back(2) → push_back(3) → push_front(0)
Дек: 
Верх -> 0 1 2 3 <- Конец
```
**Операции:**

| Операция   | Что делает       | Сложность |
| ---------- | ---------------- | --------- |
| push_front | вставка в начало | O(1)      |
| push_back  | вставка в конец  | O(1)      |
| pop_front  | удалить с начала | O(1)      |
| pop_back   | удалить с конца  | O(1)      |
**Применение:** скользящие окна, BFS с приоритетами, двусторонние очереди задач.

## **4️⃣ Очередь с минимумом (MinQueue)**

**Идея:** держим **дополнительный дек**, чтобы быстро знать минимальный элемент

```
enqueue(3) → enqueue(1) → enqueue(2)

Обычная очередь: 3 1 2
Дек minDeque:      1 2 (minDeque.front() = 1)

dequeue() → удаляем 3
Очередь: 1 2
minDeque: 1 2 → минимальный элемент = 1

dequeue() → удаляем 1
Очередь: 2
minDeque: 2 → минимальный элемент = 2
```

**Сложность операций:** O(1) амортизированно на enqueue/dequeue/getMin

**Применение:**

- Скользящее окно min/max
- Оптимизация алгоритмов типа “максимум в каждом окне размера k”
