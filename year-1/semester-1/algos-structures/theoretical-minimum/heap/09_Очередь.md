## 9. Очередь  

### Определение

**Очередь (Queue)** — это абстрактная структура данных, работающая по принципу  
**FIFO (First In — First Out)** — _первый вошёл, первый вышел._

> Пример: очередь в магазине — кто первый пришёл, тот первый обслужен.

### Основные операции

| Операция           | Описание                              | Время    | Память   |
| ------------------ | ------------------------------------- | -------- | -------- |
| `push(x)`          | добавить элемент `x` на вершину стека | **O(1)** | **O(1)** |
| `pop()`            | удалить верхний элемент               | **O(1)** | **O(1)** |
| `front()`          | вернуть верхний элемент (не удаляя)   | **O(1)** | **O(1)** |
| `empty()`          | проверить, пуст ли стек               | **O(1)** | **O(1)** |
| `size()`           | вернуть количество элементов          | **O(1)** | **O(1)** |
| `find(x)`          | найти элемент x в очереди             | **O(n)** | **O(1)** |


### Использование очереди

| Применение                  | Пример                              |
| --------------------------- | ----------------------------------- |
| Обработка задач по порядку  | планировщик процессов               |
| BFS (поиск в ширину)        | алгоритмы на графах                 |
| Буферизация данных          | ввод-вывод, потоки                  |
| Очереди событий             | обработка входных сообщений         |
| Симуляция реальных очередей | моделирование касс, клиентов и т.д. |
|                             |                                     |
